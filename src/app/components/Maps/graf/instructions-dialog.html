Instructions

1. Create Goal

Write the name of the goal in the designated area

Click Create New Goal

Select the position in the graph where the goal will be created

2. Create Strategy

Write the name of the strategy in the designated area

Click Create New Strategy

Select the source Goal on the graph and drag the mouse to the Target Goal

3. Update Element

Select a Strategy or a Goal clicking on it, the color of the selected element will slightly change

Click on Update Selected Element

Write the new Name on the designated area and click save
